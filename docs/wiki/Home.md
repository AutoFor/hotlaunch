# hotlaunch

Windows でどのアプリを使っていても、キー2回押すだけで好きなアプリを瞬時に開けるツールです。

---

## これは何をするツール？

たとえばターミナル（WezTerm）を開きたいとき、普通はタスクバーのアイコンをクリックしたり、スタートメニューを開いたりしますよね。
**hotlaunch** を使うと、`F12` を押してから `W` を押すだけでターミナルが開きます。

- すでに起動していれば、そのウィンドウが前面に出てきます
- 最小化していても、自動で元のサイズに戻ります
- まだ起動していなければ、新しく起動します

---

## 使い方

1. `hotlaunch.exe` を起動する（タスクバーのトレイに常駐します）
2. どのアプリを使っていても、**リーダーキー**（デフォルト: `F12`）を押す
3. 2秒以内に**ショートカットキー**を押す（デフォルト: `W` = WezTerm）

```
F12 → W   WezTerm を開く（または前面に出す）
```

リーダーキーとショートカットの組み合わせは自由に設定できます。

---

## 設定ファイル

`C:\Users\あなたのユーザー名\.hotlaunch\config.json` に設定が入っています。
はじめて起動したとき自動で作られます。

```json
{
  "leader": {
    "key": "F12",
    "timeoutMs": 2000
  },
  "hotkeys": [
    {
      "key": "W",
      "appPath": "C:\\Program Files\\WezTerm\\wezterm-gui.exe",
      "processName": "wezterm-gui"
    }
  ]
}
```

アプリを追加したい場合は `hotkeys` の配列に項目を追加するだけです。

---

## 終了方法

タスクバー右端のトレイアイコンを右クリック → **「終了」** をクリック
